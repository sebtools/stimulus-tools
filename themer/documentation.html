Themer Controller

Themer Controller is a Stimulus controller that discovers CSS themes declared using [data-theme="name"] rules and creates a simple UI to switch between them. It persists the selected theme to localStorage and dispatches events when the theme changes.

Usage

Attach the controller to any container element in your page. By default the controller will scan stylesheets for rules matching [data-theme="..."], create a button for each theme and place them inside the controller element. Example:

<div data-controller="themer">
	</div>

The controller will add buttons with data-theme="<name>" and a data-action wired to themer#set. When a theme button is clicked the controller sets the data-theme attribute on the nearest ancestor element that already has a data-theme or falls back to <html>.

Themes (CSS)

Define themes using attribute selectors. Each theme can expose a human-friendly label using a custom property named --theme-label. Example:

[data-theme="light"] {
	--bg: #fff;
	--fg: #111;
	--theme-label: "Light";
	}

	[data-theme="dark"] {
	--bg: #111;
	--fg: #fff;
	--theme-label: "Dark";
	}

	/* Use the variables in your site styles, e.g. */
	body { background: var(--bg); color: var(--fg); }

Attributes

data- Attributes

Attribute Element(s) Default Value(s) Description
controller (any) themer Attach this controller to a container that will hold the theme buttons.
action button click->themer#set (string) Buttons created by the controller use this action to call the set method.
data-theme button (theme name) Buttons include this attribute; when clicked the controller applies the value to the nearest theme element (or <html>). Themes are discovered by searching stylesheets for [data-theme="name"] rules.

Targets

The controller creates elements with the following target:

Target Element(s) Description
buttondiv div (container around each button) Wrapper inserted around each generated button. The wrapper receives aria-current="true" when its button matches the active theme.

Events

themer: Events

Event Parameters Description
initialized Dispatched from initialize() when the controller instance is created.
connected Dispatched when the controller is connected to the DOM.
disconnected Dispatched when the controller is disconnected from the DOM.
theme-changed

theme: (string) the name of the newly applied theme

element: (Element) the element that received the data-theme attribute

 Fired whenever the active theme is changed via the controller's UI or API.

Methods

These methods are available on the controller instance (for example by referencing document.querySelector('[data-controller="themer"]').themer).

Method Returns Arguments Event(s) Description
set event: (Event) click event from a generated button theme-changed Handler for button clicks; reads the button's data-theme and applies it.
setTheme theme: (string), element: (Element, optional) theme-changed Apply a theme programmatically. Sets the data-theme attribute on the nearest theme element (or provided element) and stores the choice in localStorage.
setThemeFromStorage element: (Element, optional) theme-changed Reads the remembered theme from localStorage (key: theme) and applies it.
getCurrentTheme string element: (Element, optional) Returns the currently applied theme (from the nearest theme element) or the default.
getDefaultTheme string Returns the default theme based on prefers-color-scheme ("dark" or "light").
getThemes Array<{name,label}> Scans loaded stylesheets for [data-theme="name"] rules and returns discovered themes with labels (from --theme-label if present).
getThemeLabel / getThemeVariable / getThemeElement (see code) Utility methods to read theme-related information from stylesheets and find the element that currently holds a theme.
updateActiveButtonDiv Updates generated button wrappers to reflect the active theme by setting aria-current="true" on the matching wrapper.

Persistence

The controller uses localStorage with the key theme to remember the last selected theme. Calling setThemeFromStorage() will reapply that value to the provided element (or the controller's element).

Notes

  • The controller ignores cross-origin stylesheets when scanning for theme rules.
  • If no theme is applied, the controller falls back to a default determined by prefers-color-scheme.
  • Buttons and wrappers are generated automatically. You can also create your own buttons using data-theme and data-action="themer#set" if you prefer.
